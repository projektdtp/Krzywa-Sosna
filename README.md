# Krzywa Sosna — strona internetowa

Folder zawiera kompletną statyczną stronę Krzywej Sosny.

## Publikacja w GitHub Pages

1. Utwórz lub otwórz repozytorium GitHub.
2. Wgraj zawartość tego folderu do katalogu głównego repozytorium.
3. Nie wgrywaj ZIP-a jako pojedynczego pliku — najpierw go rozpakuj.
4. W repozytorium otwórz Settings → Pages.
5. Wybierz publikację z gałęzi main i katalogu root.

Plik startowy strony: index.html
